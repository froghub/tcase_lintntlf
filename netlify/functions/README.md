## Сайт визитка на html+js+css+nodejs

### Запуск локально (если хочется проверить работает ли )
1. `cp .env.example e.nv`  - копируем и заполняем конфиг
2. `npm install` - устанавливаем зависимости
3. `npm netlify dev` - запускаем. в терминале будет локальный адрес, на котором запустится

### Посмотреть вживую:
1. Сама визитка доступна по адресу https://froghub.github.io/tcase_lintntlf/
2. Код (его уже смотрите,но все же) https://github.com/froghub/tcase_lintntlf
